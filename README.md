# find-file

CLI-утилита для поиска файлов на Go с фильтрацией по имени, расширению, размеру и содержимому файлов.  
Поддерживает рекурсивный обход папок, вывод номеров строк и сохранение результатов.
# Установка
Склонируйте репозиторий и соберите бинарный файл:

git clone https://github.com/sinon273/find-file.git
cd find-file
go build -o find-file

Теперь вы можете запускать утилиту через ./find-file (Linux/macOS) или find-file.exe (Windows).

# Использование

find-file [флаги] <путь>
Если не указать путь, поиск будет выполняться в текущей директории

  	              
Флаг -p <строка>	Искать файлы, содержащие подстроку в имени

Флаг -s <число>	Минимальный размер файла в байтах

Флаг -ext <расширение>	Фильтр по расширению файла (например, .go)

Флаг -content <строка>	Искать внутри файлов (выводит номер строки и текст)

Флаг -o  <файл>	Сохранить результат в указанный файл

Флаг -h,  -help	Показать справку

#Пример
find-file -content "Hello" <Путь к файлу>
